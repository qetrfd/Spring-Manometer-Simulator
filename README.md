# Spring Manometer Simulator

Interactive Python application that simulates the behavior of a **spring-loaded manometer** and calculates the mechanical properties involved in the system. The program combines **engineering calculations, graphical visualization, and a free body diagram (FBD)** to demonstrate how pressure applied to a piston generates force and compresses a spring.

The application provides a graphical interface where users can enter pressure, piston diameter, and displacement values. Using these inputs, the system calculates the **force produced by pressure and the spring stiffness constant**, while visually representing the mechanical system and the equilibrium of forces.

The project was developed as an educational engineering tool to help visualize the relationship between **pressure, force, displacement, and spring stiffness** in mechanical systems.

---

# Engineering Concepts Used

The simulation is based on fundamental mechanical engineering equations related to pressure systems and spring mechanics.

### Pressure Force

Pressure applied to the piston generates a force according to:

```
F = P · A
```

Where:

* **F** = Force applied on the piston (N)
* **P** = Pressure applied to the chamber (Pa)
* **A** = Cross-sectional area of the piston (m²)

---

### Piston Area

The piston area is calculated using the circular area formula:

```
A = πD² / 4
```

Where:

* **D** = piston diameter

---

### Spring Constant (Hooke’s Law)

Once the force is known, the spring stiffness constant is calculated using:

```
k = F / x
```

Where:

* **k** = spring constant
* **F** = force generated by pressure
* **x** = spring displacement

# Features

* Interactive graphical interface built with **Tkinter**
* Real-time calculation of mechanical properties
* Visualization of a **spring–piston system**
* Animated compression of the spring
* Free Body Diagram (FBD) showing acting forces
* Random data generator for testing scenarios
* Automatic unit conversion (user inputs vs SI units)

---

# User Inputs

The user provides three parameters:

| Parameter | Description                    | Units |
| --------- | ------------------------------ | ----- |
| P         | Pressure applied to the system | kPa   |
| D         | Piston diameter                | mm    |
| x         | Spring displacement            | mm    |

The program automatically converts values into **SI units** for accurate calculations.

---

# Calculated Results

The program calculates and displays:

* **Piston Area (A)**
* **Pressure Force (F)**
* **Spring Constant (k)**

Results are displayed both in engineering units and SI units for clarity.

---

# Visual Simulation

The interface includes a dynamic visualization showing:

* A piston inside a chamber
* A spring connected to the piston
* Pressure entering the chamber
* Spring compression animation
* A Free Body Diagram showing the equilibrium of forces

The simulation illustrates the equilibrium condition:

```
F_pressure = F_spring
```
